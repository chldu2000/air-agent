# v0.4 Multi-Model Provider Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Introduce an `LLMProvider` abstraction so `Agent` and Skills routing no longer depend directly on OpenAI response shapes, while preserving the current OpenAI-compatible user API.

**Architecture:** Add provider dataclasses/protocols that describe only the minimum Agent loop needs: completion text, tool calls, stream deltas, and token usage. Implement `OpenAIProvider` as the default adapter around `AsyncOpenAI`, then migrate `Agent` and `LLMSkillRouter` to consume provider-neutral responses. Keep non-OpenAI SDK support out of core for v0.4; custom providers and fake providers cover extension and tests.

**Tech Stack:** Python 3.11 dataclasses and protocols, `openai.AsyncOpenAI`, existing `pytest`/`pytest-asyncio` tests, no new runtime dependencies.

---

## Scope Source

This plan implements `docs/air-agent-roadmap.md` Milestone v0.4:

- Unified `LLMProvider` abstraction.
- Default OpenAI provider preserving current behavior.
- OpenAI-compatible `base_url`, `api_key`, and `default_headers` support.
- Future extension points for Anthropic, Ollama, vLLM, and local models without adding SDKs now.
- Unified normal completion, streaming, tool calling, and token usage shapes.
- `Agent` loop no longer directly depends on `AsyncOpenAI` response objects.
- Skills routing reuses the provider abstraction.
- Fake provider tests cover ReAct, streaming, tool calls, and skills routing.

```python
from __future__ import annotations

from collections.abc import AsyncIterator
from typing import Any

from openai import AsyncOpenAI

from air_agent.providers.types import (
    LLMResponse,
    LLMStreamChunk,
    LLMStreamToolCallDelta,
    LLMToolCall,
)
from air_agent.types import TokenUsage


class OpenAIProvider:
    supports_tools = True
    supports_streaming = True

    def __init__(
        self,
        *,
        client: AsyncOpenAI | None = None,
        api_key: str | None = None,
        base_url: str | None = None,
        default_headers: dict[str, str] | None = None,
    ) -> None:
        self.client = client or AsyncOpenAI(
            api_key=api_key,
            base_url=base_url,
            default_headers=default_headers,
        )

    async def complete(
        self,
        *,
        model: str,
        messages: list[dict[str, Any]],
        tools: list[dict[str, Any]] | None = None,
        **options: Any,
    ) -> LLMResponse:
        kwargs: dict[str, Any] = {
            "model": model,
            "messages": messages,
            **options,
        }
        if tools:
            kwargs["tools"] = tools
        response = await self.client.chat.completions.create(**kwargs)
        message = response.choices[0].message
        return LLMResponse(
            content=message.content or "",
            tool_calls=_map_tool_calls(message.tool_calls),
            usage=_map_usage(response.usage),
        )

    async def stream(
        self,
        *,
        model: str,
        messages: list[dict[str, Any]],
        tools: list[dict[str, Any]] | None = None,
        **options: Any,
    ) -> AsyncIterator[LLMStreamChunk]:
        kwargs: dict[str, Any] = {
            "model": model,
            "messages": messages,
            "stream": True,
            "stream_options": {"include_usage": True},
            **options,
        }
        if tools:
            kwargs["tools"] = tools
        stream = await self.client.chat.completions.create(**kwargs)
        async for chunk in stream:
            usage = _map_usage(getattr(chunk, "usage", None))
            delta = chunk.choices[0].delta if chunk.choices else None
            if delta is None:
                yield LLMStreamChunk(usage=usage)
                continue
            yield LLMStreamChunk(
                content_delta=delta.content or "",
                tool_call_deltas=_map_tool_call_deltas(delta.tool_calls),
                usage=usage,
            )


def _map_usage(usage: Any) -> TokenUsage | None:
    if not usage:
        return None
    return TokenUsage(
        prompt_tokens=usage.prompt_tokens,
        completion_tokens=usage.completion_tokens,
        total_tokens=usage.total_tokens,
    )


def _map_tool_calls(tool_calls: Any) -> list[LLMToolCall]:
    if not tool_calls:
        return []
    return [
        LLMToolCall(
            id=tc.id,
            name=tc.function.name,
            arguments=tc.function.arguments,
        )
        for tc in tool_calls
    ]


def _map_tool_call_deltas(tool_calls: Any) -> list[LLMStreamToolCallDelta]:
    if not tool_calls:
        return []
    return [
        LLMStreamToolCallDelta(
            index=tc.index,
            id=tc.id,
            name=tc.function.name if tc.function and tc.function.name else None,
            arguments=tc.function.arguments if tc.function and tc.function.arguments else "",
        )
        for tc in tool_calls
    ]
```
